<b>Markup :</b> HTML

<b>Styling :</b> CSS <b></b>

<b>Preprocessor :</b> SASS

<b>Language :</b> Javascript

<b><u>Libraries</u></b>

<b>Carousel :</b> infinite-react-carousel

<b>Icons :</b> Material-ui Icons
